The whole or a division of goods, services or works to be procured.<br/><br/>Additional Information:<br/>Anything that can specify the procurement content (i.e. goods, services, work) is a Procurement Object. <br/>In a sense, such an "object" can constitute the "object of a Contract".<br/>To test whether something is a Procedure Object check if it can have a Purpose and/or CPV classification (<i>The CPV establishes a single classification system for public procurement aimed at standardising the references used by Contracting Authorities and entities to describe the subject of Procurement Contracts.</i>).<br/><br/>Note:<br/>Procedure, seems to be an exception from this rule. Because it is a conflated term: it carries process properties and "purpose" properties. <br/>
