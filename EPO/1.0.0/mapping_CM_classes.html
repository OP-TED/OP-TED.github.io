Submit an issue:

To propose a new mapping for one of the class or to create any issue related to the mapping of the classes, please click here and then click on "New issue". For a new mapping, adapt the first column of the table proposed to follow the class template below:

Element

Description

Label

A short title of the class, e.g. “Contract”

URI

A string of characters used to identify a resource.

Use the second column of the template as guidance to propose a URI.
