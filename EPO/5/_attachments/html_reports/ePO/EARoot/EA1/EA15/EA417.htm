The Language authority table is a controlled vocabulary that lists concepts associated with languages.<br/><br/>Its main scope is to support activities associated with publication process.<br/><br/>The concepts included are correlated with the ISO 639 international standard. This standard is issued in several parts:<br/><br/>ISO 639-1 contains strictly two alphabetic letters (alpha-2),<br/><br/>ISO 639-2/B (B = bibliographic) is used for bibliographic purpose (alpha-3),<br/><br/>ISO 639-2/T (T = terminology) is used for technical purpose (alpha-3),<br/><br/>ISO 639-3 covers all the languages and macro-languages of the world (alpha-3); the values are compliant with ISO 639-2/T.<br/><br/>If an authority code is needed for a language without assigned ISO code, an alphanumeric code is created in order to not confuse with ISO codes that are strictly alphabetic.<br/><br/><br/>
