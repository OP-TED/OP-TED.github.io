Details:
The awarded value of all lots announced in this notice, including options and renewals. <br/><br/>Additional information:<br/>The values of the individual Lots awarded under a framework agreement and mentioned in this notice are included.<br/><br/>The values of the individual lots announced in a CAN are to be traced back and added to provide this value.<br/><br/>This corresponds to the BT-161 in eForms.<br/><br/>WG Approval 03/12/2019<br/><br/>

Details:
The maximum value which can be spent through all the framework agreements announced in this notice, including options and renewals of contracts.<br/><br/>Additional information:<br/><br/>The value provided is a threshold value that implicity means that it cannot be exceeded however it may not be reached during the execution of a contract.<br/><br/>The Framework Agreements in a CAN are to be traced back and added to provide this value.<br/><br/>This corresponds to the BT-118 in eForms.<br/><br/>WG Approval 03/12/2019<br/>
