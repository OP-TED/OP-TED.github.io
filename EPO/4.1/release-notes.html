eProcurement Ontology v4.1.0-rc.2

The release v4.1.0-rc.2 and its corresponding technical files can be found here.

Release notes

  • eAccess module was released.

  • eSubmission module was released.

  • Updates for the following modules were made:

    • ePO core

    • eCatalogue

    • eOrdering

    • eFulfilment

    • eContract

    • eNotice.

  • The (closed) github issues justifying these updates can be found here
