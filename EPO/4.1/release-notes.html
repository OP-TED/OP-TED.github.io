eProcurement Ontology v4.1.0

The release v4.1.0 and its corresponding technical files can be found here.

Release notes

30 July 2024

  • eAccess module was released.

  • eSubmission module was released.

  • Updates for the following modules were made (under Change notes) :

    • ePO core

    • eCatalogue

    • eOrdering

    • eFulfilment

    • eContract

    • eNotice.

  • The (closed) github issues justifying these updates can be found here
