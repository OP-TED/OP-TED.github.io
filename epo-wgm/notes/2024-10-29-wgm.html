Working Group meeting

Date: 29/10/2024  
Participants: Ioannis Fountoukidis, Natalie Muric
Model editor: Achilles Dougalis
Note editor: Grzegorz Kostkowski

Agenda

  • Work on PEPPOL T009 and T010 transactions: Incorporate Tender Clarification Request and Tender Request into ORSD Evaluation document.

Discussions

  • The document was updated and extended to accommodate for Tender Clarification Request and Tender Request. The scope of the changes included use cases, user stories, natural language sentences.

  • New natural language sentences were created based on PEPPOL requirements defined in T009 and T010. The sentences were adapted to match the Evaluation model. For example, Tenderer was used instead of Economic Operator.

  • The exemplary evaluation report was analysed to determine what data needs to be modelled.

  • It was emphasised that the designed model must capture commonalities, despite variations in evaluation procedures across EU countries.

  • More complex example was found and shared with the WG participants: https://www.anticorruzione.it/amministrazione-trasparente/bandi-di-gara-e-contratti/gare-in-corso/bando-di-gara-del-26-giugno-2023. Some of its key parts were explained. The below table in Documenthttps://www.anticorruzione.it/documents/91439/129009/Verbale+n.+7+del+29.9.2023+Commissione+giudicatrice.pdf/11ea5002-9721-156b-0f43-7396537a6465?t=1697465711923[ number 7] presents an example case where a Clarification Request is submitted.

  • Results of a technical assessment are presented in the second column and an economical score in the third column.

  • Economical score is based on the offered cost.

  • The highlighted first row contains a suspicious case where a request for clarification can be submitted to ask the Tenderer to justify its statements (as it claims outstandingly high performance for unusually low cost).

  • The further part of the document explains that the Tenderer provided the response that was analysed and it was concluded that the offer is valid.

Action Points

  • WG will start modelling activities in the next meeting.
