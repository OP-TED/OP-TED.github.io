eCatalogue meeting

Date: 16/06/2022
Participants: Veit Jahns, Natalie Muric, Giampaolo Sellitto
Model editor: Eugeniu Costetchi
Note editor: Andreea PasÄƒre

Discuss GH 284

  • In ePO, the versioning is done at the document level.

  • Need to decide whether to use action-code or change-type controlled list:

    • For a type of change, different systems may have different actions

    • Indicating actions might be easier to implement

  • There are two options:

    • We can create an epo-ecat:Activity class that inherits from prov:Activity and epo-ecat:influencesEntity will be linked to epo-ecat:Catalogue (or CatalogueItem) and we provide only the epo-ecat:action-code controlled vocabulary

    • We can provide the change type directly linked to the Catalogue.

  • Do we need to take into account the temporal aspect?

  • Another option is to have a subclass of Catalogue named CatalogueFragment who holds just the part that needs to be transmitted and indicates which action can be done.

  • Need to further discuss the name of the controlled vocabulary to be used (action-code vs. operation-code or activity-code)
